# Threelite Bash

A browser-based pixel boss game starring Norm.

## Play Locally

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000/`.

## Deploy

This is a static site. Deploy the repository root to GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any static file host.

For GitHub Pages:

1. Create a repository named `threelite-bash`.
2. Push these files to the repository.
3. In GitHub, open Settings > Pages and publish from the main branch root.
